## 職務経歴

### 2019/11 - 現在 : 株式会社ファイブスターインタラクティブ  
Webエンジニアとして、コーポレートサイト・LPのコーディング業務や、当時社内に本格的なノウハウがなかったLaravelを用いて、大手学校法人のポータルサイト開発を担当しました。業務を経て、獲得した経験や知識を社内で共有したいと考え、チームリーダーと共に社内ナレッジツール（DocBase）の活用を会社へ提言し、導入しました。

#### 社内ナレッジツール DocBase の導入 2022/4 - 現在

##### プロジェクト概要
社内ナレッジツールである[DocBase](https://docbase.io/)の活用をチームリーダーと協力し、会社へ提案・導入して頂きました。

DocBase の導入前は、以下のような問題がありました。

- 受託企業かつ案件ごとに人員が割り当てられるため、業務から得られる知識に差が大きく出る
- 個人の力量が上がるが、チーム全体の底上げにならない
- 経験した人が万が一退職した場合、知見も一緒になくなってしまう
- デザインとコーディングを分業しているため、他分野に関する知識が乏しくなりコミュニケーションエラーが多い。
- 会社の人数も増えてきたため、他の部署が何をしてるのか知らないことが多い。

これらの課題を解決するためには、各々が得た知見を共有しためていくための仕組みが必要だと考え、DocBase の導入提案に踏み切りました。

導入前のテスト期間当初は、なかなか DocBase で記事を書く人も少なく、記事数が増えない状況でした。そこで、もともと社内チャットツールに存在していた、技術情報共有ルームから過去の内容を転機するなどして、記事数を増やしていきました。

記事数が増えると徐々に社内でも記事を書く人が増えていき、1ヶ月で記事数が50件を超えたところで、社内での本導入が決まりました。
現在もデザイナー・ディレクター含め、さまざまな知見を日々共有し合い、着実に社員の成長に貢献できていると実感しております。

#### 学校法人ポータルサイト開発 2021/8 - 現在
